📌 Project Title

Sales Data Visualization Dashboard

📖 Description

This project focuses on transforming cleaned sales data into meaningful visual insights using Microsoft Excel. The aim is to present data in an easy-to-understand format through charts and dashboards, helping identify patterns, trends, and business insights.

🎯 Objective

Convert raw data into visual representations
Identify trends and patterns in sales data
Present insights clearly for decision-making
Build an interactive and structured dashboard

🛠️ Tools & Techniques Used

Microsoft Excel
Pivot Tables
Data Visualization (Bar Charts & Pie Charts)
Dashboard Design

📊 Visualizations Included

🔹 Product-wise Sales Quantity
Bar chart representing quantity sold for each product
Helps identify high-demand products
🔹 Revenue by Product
Bar chart showing total revenue generated by each product
Highlights top-performing products
🔹 Payment Method Distribution
Pie chart displaying different payment methods used by customers
Helps understand customer preferences
🔹 Coupon Usage Distribution
Pie chart representing usage of coupon codes
Helps analyze effectiveness of promotional offers

📌 Key Insights

Chairs have the highest demand and generate the highest revenue
Online payment is the most preferred payment method
Coupon usage is fairly distributed across different types
Sales data shows variability with some high-value transactions

📊 Dashboard Overview

All visualizations are combined into a single Excel sheet to create a structured dashboard for better understanding and presentation of data.

📈 Conclusion

This project demonstrates how effective data visualization can simplify complex datasets, reveal insights, and support data-driven decision-making.
